# Space Invaders Clone

A browser game inspired by the original Space Invaders (non-ASCII, pixel-rendered on canvas).

## Controls

- `ArrowLeft`: Move left
- `ArrowRight`: Move right
- `ArrowUp`: Fire

## Gameplay

- Classic 5x11 alien formation that marches, drops, and speeds up.
- Destructible shields.
- Alien projectiles and player lives.
- Mystery UFO bonus target.
- Score, best score, and level progression.

## Run

1. Open `index.html` in your browser.
2. Click **Restart** anytime to start a fresh run.
